Numba CUDA MLIR
===============

Numba CUDA MLIR is an evolution of Numba-CUDA that improves upon its technical
foundation and performance to provide the future evolution of CUDA Python JIT
compilation.

It is used for writing SIMT kernels in Python, for providing Python bindings
for accelerated device libraries, and as a compiler for user-defined functions
in accelerated libraries like `RAPIDS <https://rapids.ai>`_.

* To install Numba CUDA MLIR, see: :ref:`numba-cuda-installation`.
* To get started writing CUDA kernels in Python, see :ref:`writing-cuda-kernels`.
* Browse the :ref:`numba-cuda-examples` to see a variety of use cases of
  Numba CUDA MLIR.
